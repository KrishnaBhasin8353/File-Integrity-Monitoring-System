# 🔐 Secure File Integrity Monitoring System (FIMS)

## 📌 Overview
The Secure File Integrity Monitoring System (FIMS) is a Python-based cybersecurity tool that detects unauthorized file modifications, deletions, and additions within a monitored directory using SHA-256 hashing.

This project demonstrates practical implementation of file integrity verification techniques used in real-world security systems.

---

## 🚀 Features

- SHA-256 based file hashing
- Folder-wide scanning
- Detects:
  - Modified files
  - Deleted files
  - Newly added files
- JSON-based hash storage
- Activity logging with timestamps

---

## 🛠 Technologies Used

- Python
- hashlib
- os
- json
- datetime
